# AquaMeter Home Assistant Integration

Official Home Assistant integration for AquaMeter water monitoring devices.

AquaMeter is a Bluetooth water monitoring system for campervans, motorhomes, boats and other autonomous water systems.

## Status

Development in progress.

## Planned features

### v1.1.0 — Monitoring

- Water remaining, L
- Tank level, %
- Flow rate, L/min
- Today's consumption, L

### Future versions

- Basic controls: tank capacity, reserve level, full tank button
- 30-day consumption history

## Requirements

- AquaMeter device
- Home Assistant with Bluetooth support
- Raspberry Pi Bluetooth, USB Bluetooth adapter, or ESPHome Bluetooth Proxy

## Not included

For safety and reliability, the Home Assistant integration will not support:

- Firmware updates
- OTA
- Calibration
- Advanced device setup

These functions remain available only in the official AquaMeter mobile apps.
